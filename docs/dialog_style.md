# Daydream Dialog Style Guide

Comedy-forward slice-of-life. Every NPC line earns its existence.

## Hard rules

1. **Cutscene cap:** 12 textboxes max. Longer → split or cut.
2. **If a line doesn't earn a smile OR move the plot, cut it.**
3. **Tutorials are environmental.** Use signs, not lectures.
4. **Prof. Mim explains nothing twice.**
5. **Every joke is load-bearing.** It should also deliver info.

## Voice cheat sheet

| Character       | Voice                                               |
|-----------------|-----------------------------------------------------|
| Prof. Mim       | Tired, dry, no patience for ritual.                 |
| Tibb (rival)    | Earnest, sleepy, accidentally insightful.           |
| Director Hush   | Deadly sincere believer in rest.                    |
| Champion Vex    | Union-core deadpan. Will negotiate.                 |
| The Innkeeper   | Auntie energy. Food = love.                         |
| Patty (baker)   | Overcaffeinated. Sentences end mid-breath.          |

## Template: intro rewrite

**BEFORE** — 18 textboxes of exposition.
**AFTER** — 4 textboxes:

```
PROF_MIM: "You're late."
PROF_MIM: "The good starters got picked yesterday. Box's by the door."
PLAYER:   (picks)
PROF_MIM: "Try not to nap on Route 1. Try."
```

## When to break the rules

- The horror arc uses **silence**, not jokes. Leave textboxes short and weird.
- Boss confrontations may exceed 12 boxes *if* every line is a punchline or a reveal.
